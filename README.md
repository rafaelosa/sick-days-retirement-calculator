# Lemoney - מחשבון קיבוע זכויות

מחשבון אינטראקטיבי לחישוב **קיבוע זכויות בפרישה** לפי סעיף 9א לפקודת מס הכנסה (נוסחת השילוב).

## הרצה

פתח את `index.html` בכל דפדפן מודרני - זהו.

- ללא התקנות
- ללא שרת
- ללא ספריות חיצוניות
- ללא חיבור לאינטרנט

## תכונות

- חישוב מלא של נוסחת הקיזוז ב-5 שכבות בדידות
- תמיכה במספר בלתי מוגבל של מענקים והיוונים
- מידוד אוטומטי לפי מדד המחירים לצרכן (817 חודשי מדד מוטמעים)
- חישוב תקרת 32 שנות עבודה
- תחזית קצבה פטורה לשנים עתידיות (עד 2028)
- ממשק עברית RTL

## אימות

המנוע נבדק מול 14 ערכים מאקסל ייעודי וחזר תוצאות זהות (הפרש < 10⁻¹²).

## מבנה הקוד

הקובץ עצמאי לחלוטין ומכיל:
- שכבה 1: שליפת מדדים ופרמטרי שנת פרישה
- שכבה 2: חישובים פר-מענק (מידוד וקנס 1.35)
- שכבה 3: אגרגציה והגבלת תקרת 32 שנה
- שכבה 4: חישוב יתרות הון
- שכבה 5: המרה לקצבה חודשית ושיעור פטור
