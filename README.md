מחשבון אינטרנטי בעברית (RTL) לחישוב אומדן מענק ימי מחלה בפרישה לעובדי מדינה, על פי תקנות הפרישה והוראות התקשי״ר.

פותח על ידי **אלפי ביטוח ופיננסים** — מומחי פרישה וטופס 161.

## תכונות עיקריות

- **חישוב אוטומטי** של תקופת עבודה, ימי מחלה שנצברו, אחוז ניצול ומדרגת זכאות
- **שלוש מדרגות זכאות** לפי הוראות התקשי״ר (8/30, 6/30, ללא זכאות)
- **חישוב כפול**: לפי שכר חודשי או שכר יומי (אופציונלי)
- **ולידציה חכמה**: זיהוי טעויות נפוצות בהזנת תאריכים (לדוגמה: תאריך לידה במקום תחילת העסקה)
- **הדפסת דוח מקצועי** עם מיתוג מלא של החברה ופרטי העובד
- **שליחת נתונים במייל** דרך תוכנת הדואר של המשתמש
- **אישור מפורש**: שתי תיבות סימון חובה לפני ביצוע החישוב (תקנון + מסירת פרטים מרצון)
- **עיצוב רספונסיבי** מותאם למחשב ולנייד
- **ללא תלויות**: קובץ HTML יחיד, ללא צורך בשרת או npm

## העלאה ל-GitHub Pages — מדריך שלב אחר שלב

### אם אין לך עדיין GitHub:
1. היכנס ל-[github.com](https://github.com) והירשם בחינם
2. אמת את כתובת המייל שלך

### יצירת המאגר (Repository):
1. בעמוד הבית של GitHub, לחץ על **New repository** (כפתור ירוק)
2. תן שם למאגר, לדוגמה: `sick-days-calculator`
3. סמן **Public** (חובה כדי להפעיל GitHub Pages חינם)
4. **אל** תסמן את "Add a README file" (כי כבר יש לך אחד בקבצים)
5. לחץ **Create repository**

### העלאת הקבצים:
1. במסך החדש, לחץ על **uploading an existing file**
2. גרור את ארבעת הקבצים האלה:
   - `index.html`
   - `README.md`
   - `.gitignore`
   - `LICENSE.txt`
3. בתחתית, ב-Commit message, רשום: `Initial commit`
4. לחץ **Commit changes**

### הפעלת GitHub Pages:
1. במאגר שלך, לחץ על **Settings** (בתפריט העליון)
2. בתפריט הצדדי, לחץ על **Pages**
3. תחת "Source", בחר **Deploy from a branch**
4. תחת "Branch", בחר **main** ולחץ **Save**
5. המתן 1–2 דקות

### הכתובת הסופית:
האתר יהיה זמין בכתובת:
```
https://[שם-המשתמש-שלך].github.io/sick-days-calculator/
```
